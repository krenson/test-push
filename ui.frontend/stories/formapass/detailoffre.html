Détail d'une offre de prestation

Offre de prestation n°39599 : TEST ELAO : identification du niveau linguistique pour une entrée en formation ou immersion linguistique

Précision de l'intitulé :EVALUATION LINGUISTIQUE ALLEMAND ANGLAIS NEERLANDAIS

Statut : Actif

Général

Objectif : Identifier mes compétences

POUR QUI
Public cible : Demandeur d’emploi inscrit au Forem
Pré-requis : Sans objet
Compétence(s) visée(s) : A l’issue du test, le candidat connaîtra son niveau linguistique de néerlandais, anglais ou allemand, positionné sur le cadre européen commun de référence pour les langues (CECRL), en vue d’une entrée en formation ou d'une immersion à l’étranger via le Forem.

Niveau acquis : Sans Objet
Titre délivré : Sans objet

Organisation

PAR QUI
Organisme : Le Forem Luxembourg - Direction générale Formation
Type d'organisme : FOREM Office wallon de la formation professionnelle et de l'emploi
Conditions administratives d'admission :
Etre inscrit au Forem comme demandeur d'emploi

Description

Description de l'action
Description : Le test comprend les modules suivants :
• Test de langue (informatisé et oral)
• Accompagnement linguistique dans le cadre d’un projet professionnel

A l'issue du test un rapport ELAO ou visa langues est délivré.
Outils pédagogiques : • Test informatisé ELAO du Forem
